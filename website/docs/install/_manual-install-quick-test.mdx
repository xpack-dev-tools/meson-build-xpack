### Quick test {#manual-install-quick-test}

To check if the xPack Meson Build installed manually starts properly, use something like:

<Tabs groupId="operating-systems">
<TabItem value="windows" label="Windows" default>

<CodeBlock language="console"> {
`C:\\> %USERPROFILE%\\AppData\\Roaming\\xPacks\\meson-build\\xpack-meson-build-${ customField('upstreamVersion') }-${ customField('xpackSubversion') }\\bin\\TODO.exe --version
TODO ${ customField('upstreamVersion') }`
} </CodeBlock>

</TabItem>
<TabItem value="macos" label="macOS">

<CodeBlock language="console"> {
`$ ~/Library/xPacks/meson-build/${ customField('upstreamVersion') }-${ customField('xpackSubversion') }/bin/TODO --version
TODO ${ customField('upstreamVersion') }`
} </CodeBlock>

</TabItem>
<TabItem value="linux" label="GNU/Linux">

<CodeBlock language="console"> {
`$ ~/.local/xPacks/meson-build/xpack-meson-build-${ customField('upstreamVersion') }-${ customField('xpackSubversion') }/bin/TODO --version
TODO ${ customField('upstreamVersion') }`
} </CodeBlock>

</TabItem>
</Tabs>
